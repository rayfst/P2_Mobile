
### Descrição

Mini Cadastro de Tarefas Profissionais desenvolvido em Flutter com uso de banco de dados local utilizando SQFLITE e Path_provider. O aplicativo realiza um CRUD completo (inserir, listar, editar e excluir tarefas), com implementação de prioridade e filtro, utilizando os requisitos individuais fornecidos pelo professor.
